# Módulo POCA - Mercado de Previsões Capixaba

Este módulo integra a plataforma **POCA** ao ecossistema Avelarsys.

## Estrutura
- `/backend`: APIs e lógica de integração com o sistema Avelar.
- `/frontend`: Interface do usuário (React/Next.js).
- `/contracts`: Smart Contracts (Solidity) para apostas P2P e mercados de previsão.
- `/docs`: Documentação técnica e guias de implementação.

## Objetivo
Prover uma plataforma descentralizada de previsões e desafios P2P focada no Espírito Santo, utilizando a infraestrutura financeira e de usuários da Avelarsys.
