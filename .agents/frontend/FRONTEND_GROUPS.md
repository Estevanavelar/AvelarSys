# Skills de Front-end — Grupos e Descrição

Este documento descreve cada grupo de habilidades na pasta `frontend` e quando usar cada skill.

---

## 1. Top tier — Uso diário (React / Next / TypeScript)

Skills para usar ao escrever, revisar ou refatorar qualquer app React/Next/TypeScript.

| Skill | Descrição |
|-------|-----------|
| **react-best-practices** | Guia Vercel de performance em React e Next.js: eliminação de waterfalls, otimização de bundle, RSC, SWR, re-renders, hydration. Dezenas de regras priorizadas. Use ao escrever ou revisar componentes e páginas. |
| **ui-ux-pro-max** | Design: estilos, paletas, tipografia, diretrizes de UX, acessibilidade e stacks (React, Next, Vue, Svelte, Tailwind, shadcn/ui). Use ao definir ou revisar UI, landing pages e dashboards. |
| **frontend-dev-guidelines** | Padrões de front-end para React + TypeScript: Suspense-first, arquitetura por features, MUI v7, TanStack Router. Define como o código de front deve ser escrito. |
